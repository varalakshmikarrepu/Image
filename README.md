# Image Description Q&A App

## Description
This project is a web application that allows users to upload an image and ask a question like *“What is in this image?”*. The system analyzes the uploaded image and returns an automatic image description.

The app uses an image captioning model to generate a textual description of the image.

## Features
- Upload an image through a web interface
- Ask a question about the image
- Automatically generates an image description
- Simple and user-friendly interface
- Runs locally using Streamlit

## Technologies Used
- Python
- Streamlit
- Hugging Face Transformers
- PyTorch / TensorFlow
- Pillow (PIL)
